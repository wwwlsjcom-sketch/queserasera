# Exhibit Router

Use this router after selecting the report template. Select modules by analytical need and data shape. Do not use a module only because it appears in a default list.

## Data Shape Triggers

- Time series, adoption, revenue, market size, KPI trajectory -> `market-trend-index`, `small-multiples`
- Category comparison, ranking, benchmark -> `ranking-bar`, `competitor-positioning`, `grouped-bar`
- Part-to-whole, mix, share, contribution -> `stacked-share`, `value-chain`, `unit-economics-bridge`
- Before/after, margin bridge, cost stack -> `waterfall-bridge`, `unit-economics-bridge`
- Survey, sentiment, Likert, positive/negative -> `grouped-bar`, `stacked-share`, `ranking-bar`
- Risk, probability, impact, control maturity -> `risk-heatmap`, `scenario-grid`, `watch-point-tracker`
- Causal structure, dependencies, ecosystem -> `driver-tree`, `causal-map`, `ecosystem-map`
- Portfolio, market attractiveness, product options -> `opportunity-matrix`, `market-map`, `feature-priority-matrix`
- Funnel, conversion, adoption journey -> `funnel-analysis`, `customer-journey`, `market-trend-index`
- Execution plan, milestones, owners -> `implementation-roadmap`, `watch-point-tracker`, `option-scorecard`
- Investment thesis, diligence, decision gates -> `thesis-scorecard`, `decision-brief`, `watch-point-tracker`

## Selection Rules

- Normal reports: select 5-8 modules.
- Data-heavy reports: select 8-12 modules and at least five distinct visual types.
- Every major section should usually have at least one exhibit or structured evidence module.
- If a report has many tables, convert pattern-recognition tables into charts and keep lookup tables compact.
- Keep labels concise to avoid overflow, but put interpretation, evidence, and caveats in nearby narrative panels.

## Required Mix For Complete Reports

Include when the material supports it:

- At least one relationship/structure exhibit.
- At least one trend/comparison exhibit.
- At least one risk/priority/decision exhibit.
- At least one action/roadmap/watch-point exhibit for decision-oriented reports.
